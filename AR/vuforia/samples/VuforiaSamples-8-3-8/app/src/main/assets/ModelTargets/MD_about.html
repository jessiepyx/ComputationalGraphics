The Model Targets sample shows how to detect a 3D object and render a simple 3D representation over it. The sample demonstrates how Model Targets (MT), MT Advanced, and MT Advanced 360 Datasets work.

Key Functionality

  • Model Targets: Loads a single Model Target with multiple Guide Views that can be cycled to pick which view to detect of the physical model
  • MT Advanced: Loads multiple Model Targets that have automatic detection and snapping of Guide Views to to the physical models
  • MT Advanced 360: Loads a Model Target that can be detected from arbitrary views without the need for a Guide View
  • Automatic 3D object tracking after successful detection
  • Device Tracker when the object is not visible in the camera view
  • Manage camera functions: continuous autofocus

Physical Targets

  • Model Target: Mars Lander 3D Printed Model, two views
  • Model Target: Motorcycle Toy, two views

Instructions

  • Point camera at 3D object to start tracking
  • Single tap to focus
  • Double tap to access the options menu
  • Access a different dataset via the options menu
  • Reset to start over the experience
  • Mars Lander is in all three datasets
  • Motorcycle is in the MT Advanced dataset
  • Change the detection position if needed
